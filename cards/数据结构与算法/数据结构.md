# 堆

问题 | 答案 | 标签

堆常见分哪两种？ | 大顶堆/小顶堆。

优先级队列自定义比较器怎么写？ |

链表节点小顶堆写法：

```C++
auto cmp = [](ListNode* a, ListNode* b) {
    return a->val > b->val;
};

priority_queue<ListNode*, vector<ListNode*>, decltype(cmp)> pq(cmp);
```

普通 int 的比较器写法：

```C++
class cmp {
public:
    bool operator()(int &a, int &b) {
        return a > b;
    }
};

priority_queue<int, vector<int>, cmp> pq;
```

# 链表

问题 | 答案 | 标签

链表对第 n 个节点操作的基本原则是什么？ |

对第n个节点的操作，需要通过n-1个节点进行。

链表增删查的基本操作是什么？ |

增：头部

删：操作需要删除的前一个节点

查：遍历 `while(p != nullptr)`

什么时候需要断开节点和原链表之间的链接？ |

如果我们需要把原链表的节点接到新链表上，而不是 new 新节点来组成新链表的话，那么断开节点和原链表之间的链接可能是必要的。

# STL

问题 | 答案 | 标签

erase 和 remove 配合删除指定元素怎么写？ |

```C++
lists.erase(remove(lists.begin(), lists.end(), nullptr), lists.end());
```

这里erase和remove配合可以删掉指定的元素。

remove负责把需要删掉的元素放到容器的末尾，并返回指针，erase从指针开始删元素。

# 数组

问题 | 答案 | 标签

差分数组适合什么场景？ | 针对需要经常操作数组区间的情况。

差分数组怎么做区间加？ |

`num[i] = num[i-1] + diff[i]`

对于 `[l,r,val]` 操作：

`diff[l] += val`

`diff[r+1] -= val`

差分数组怎么构造结果数组？ |

`ans[0] = diff[0];`

`ans[i] = ans[i-1] + diff[i];`

前缀和适合什么场景？ | 针对需要快速进行区间查询的情况。

二维数组常见操作是什么？ | 序列化操作。

怎么在一个序列中查找另一个序列？ |

```C++
#include <algorithm>
auto it = search(a.begin(), a.end(), b.begin(), b.end());
```

在序列a中寻找序列b，找得到返回起点迭代器，找不到返回a.end()。

适用于所有容器。

vector 删除区间和删除所有某个数值怎么写？ |

删除 `[l,r)` 区间：

```C++
v.erase(v.begin() + l, v.begin() + r);
```

删除所有某个数值：

```C++
v.erase(remove(v.begin(), v.end(), val), v.end());
```

erase-remove惯用法。

vector 插入单个元素、一段序列、多个相同元素怎么写？ |

插入单个元素：

```C++
v.insert(v.begin() + pos, x);
```

插入一段序列：

```C++
v.insert(v.begin() + pos, b.begin(), b.end());
```

插入多个相同元素：

```C++
v.insert(v.begin() + pos, k, x);
```

数组排序怎么写？ |

从小到大排序：

```C++
ranges::sort(nums);
```

按二维数组第一列排序：

```C++
sort(intervals.begin(), intervals.end(), [](vector<int>& a, vector<int>& b) {
    return a[0] < b[0];
});
```

数组求和 reduce 和 accumulate 怎么写？ |

```C++
int s = reduce(cardPoints.begin(), cardPoints.begin() + m);
```

需要：

```C++
#include <numeric>
```

```C++
long long sum = accumulate(nums.begin(), nums.begin() + 2 * k, 0LL);
```

这里开始的数值如果不填写0LL，sum默认还是int类型，某些情况下会超过题目要求。

# 字符串

单个字符怎么变小写？ |

```C++
char f;
f = tolower(f);
```

string 删除子串怎么写？ | s.erase(pos, len)

string 插入子串怎么写？ | s.insert(pos, "abc");

string 查找、统计和判断字符是否存在怎么写？ |

查找子串：

```C++
int pos = s.find("abc");
```

统计字符串中指定字母的数量：

```C++
int k = ranges::count(s, '1');
```

查找字符串中是否存在字母：

```C++
s.find('a') == string::npos
```

等于就是没找到，不等就是找到了。
