怎么判断一个数是不是 2 的幂次？ |

`n & (n - 1) == 0`

如果是2的幂次，就是10000这种形式，减掉1之后会是01111，与上肯定为0。

反之如若不是2的幂次，肯定减掉1之后还有1，不会完全是0。

# 背包问题

问题 | 答案 | 标签

怎么判断一道题是不是背包问题？ |

从数组里挑一些元素，让它们的和满足某种条件，基本就是背包问题。

也就是选一些数，凑成一个目标值。

背包问题建模时先问哪几个问题？ |

1.物品是什么？

2.背包重量是多少？

3.每个物品可以放几次？

4.求最大值/最小值/方案数？

背包问题除了最大价值还能解决什么？ |

背包问题不仅可以解最大价值，也可以解是否装满。

只需要最大装的等于容量就可以了。

最大化最小值,最小化最大值分别怎么用二分做？| 这是要求性质边界，也就是满足性质的最小边界。

```C++
while(le<ri)
   {
    int mid = (le+r+1)/2;//求可行答案最大值,向上取整数
    if(check(mid))
    {
        le = mid;//mid可行,答案在[mid,r]
    }
    else ri=mid-1;//mid不可行,答案在[le,mid-1]
   }
```

```C++
while(l<r)
  {
    int mid =l + (r-l)/2;//求可行答案最小值,向下取整数
	if(check(mid))
    {
      r= mid;//mid可行,答案在[l,mid]
    }
    else l=mid+1;//mid不可行,答案在[mid+1,r];
  }
```
