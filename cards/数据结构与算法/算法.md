## 背包问题

问题 | 答案 | 标签

怎么判断一道题是不是背包问题？ |

从数组里挑一些元素，让它们的和满足某种条件，基本就是背包问题。

也就是选一些数，凑成一个目标值。

背包问题建模时先问哪几个问题？ |

1.物品是什么？

2.背包重量是多少？

3.每个物品可以放几次？

4.求最大值/最小值/方案数？

背包问题除了最大价值还能解决什么？ |

背包问题不仅可以解最大价值，也可以解是否装满。

只需要最大装的等于容量就可以了。

0 1 背包问题是哪一类问题？ | 在体积限制下，每个物体只可以选一个

完全背包问题是哪一类问题 ? | 在体积限制下，每个物品可以选无限个

0 1 背包问题的递推公式是什么？ |`f[i][j] = max(f[i-1][j],f[i-1][j-v]+w)`,0 1 背包问题都是从i-1层转移过来的

完全背包问题，朴素做法的递推公式是什么？优化后的递推公式是什么？（只需要枚举两个状态） |

朴素的递推公式是`f[i][j]=f[i-1][j-v[i]*k]+w[i]*k `，k是去枚举选多少个

```C++
f[i][j] = max(f[i-1][j],f[i-1][j-v]+w,f[i-1][j-2v]+2w......)
f[i][j-v] =max(f[i-1][j-v],f[i-1][j-2v]+w...)//相比于前面只是每一项少了w
//所以可以得到优化后的递推公式
f[i][j] = max(f[i-1][j],f[i][j-v]+w);//可以减低k的枚举，时间复杂度变到平方级别,完全背包这里是从第i层转移过来的
```

 完全背包问题的一维写法怎么写？ |

```C++
for(int i=1;i<=n;i++)
  {
    	for(int j=v[i];j<=m;j++)
          f[j] = max(f[j],f[j-v[i]]+w[i]); //更小体积的已经尝试过了，可以再次利用数据，实现无限次选择
  }
cout<<f[m]<<endl;
```

多重背包问题是哪一类问题？ | 每一个物品可选的数量是有不固定限制的，可以选`s[i]`件

多重背包问题的朴素版本怎么做递推？|

```C++
for(int i=1;i<=n;i++)
  {
    for(int j=0;j<=m;j++)
      {
        for(int k = 0;k<=s[i]&&k*v[i]<=j;k++)
          {
            f[i][j] = max(f[i][j],f[i-1][j-v[i]*k]+w[i]*k);
          }
      }
  }
```

多重背包问题的优化是怎么做的 ,优化后的时间复杂度是多少？ | 通过二进制优化，将S个物品拆分为logS组，每组看成是一个新的物品，任何正整数s都可以表示为若干个2的幂次之和。拆完之后就可以用一维优化的0 1 背包问题去解。时间复杂度是O(nmlogs)

```C++
//二进制拆分
int cnt =0; //记录拆分后的物品数量
for(int i=1;i<=n;i++)
  {
    int a,b,s;//体积，价值，数量
    cin>>a>>b>>s;
    int k=1;
    while(k<=s)
      {
        cnt++;
        v[cnt] = a*k; //新物品的体积
        w[cnt] = b*k; //新物品的价值
        s-=k;
        k*=2;
      }
	//存剩余部分
      if(s>0)
      {
		cnt++;
        v[cnt] = a*s;
        w[cnt] =b*s;
      }
  }
// 01 背包求解部分
n = cnt;
for(int i=1;i<=cnt;i++)
  for(int j=m;j>=v[i];j--)
    f[j]=max(f[j],f[j-v[i]]+w[i]);
```

分组背包问题怎么解？ | 分组背包问题是每一组只能选一个物品，所以枚举就好，然后用一维优化的0 1 背包问题求解

```C++
    int n,m;
    cin>>n>>m;
    for(int i=1;i<=n;i++)
    {
        cin>>s[i];//存储第i组物品的个数。
        for(int j=1;j<=s[i];j++)
        {
            cin>>v[i][j]>>w[i][j];//第i组第j个物品的价值和体积。
        }
    }
    for(int i=1;i<=n;i++)
    {
        for(int j=m;j>=0;j--)
        {
            for(int k=1;k<=s[i];k++)
            {
                if(v[i][k]<=j)f[j] = max(f[j],f[j-v[i][k]]+w[i][k]);
            }
        }
    }
    cout<<f[m];
```
