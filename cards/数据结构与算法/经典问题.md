二叉树题目应该怎么拆？ | 把题目要求细化成每个节点需要做的事情。

二叉树的最近公共祖先怎么求？ | 通过递归查找，

```C++
class Solution {
public:
    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
        if(!root||root==p||root==q)return root; //这里很重要，当p或者q本身就是最近公共祖先的时候这里保证了可以返回正确的节点。
        TreeNode* l = lowestCommonAncestor(root->left,p,q);
        TreeNode* r = lowestCommonAncestor(root->right,p,q);
        if(l!=nullptr&&r!=nullptr)return root;
        if(l!=nullptr)return l;
        if(r!=nullptr)return r;
        return nullptr; 
  
    }
};
```

# 二分

问题 | 答案 | 标签

什么题目可以用二分法做？| 有单调性的题目，重点就是寻找题目答案的单调性

对于给定的一个长度为 **N** 的正整数数列 **A**1∼**N**，现要将其分成 **M**（**M**≤**N**）段，并要求每段连续，且每段和的最大值最小。数据量给到1e5怎么解 ？| 关键是识别出答案可以二分，如果最大段和限制为x可以成功，那么最大段和限制为更大的x也可以成功。


```C++
#include<bits/stdc++.h>
using namespace std;
const int N = 1e5+10;
int n,m;
int num[N];
//检查最大段和x是否可以划分出来，用贪心法划分
bool check(long long x)
{
    int d = 1;
    long long sum =0;
    for(int i=1;i<=n;i++)
    {
        if(num[i]>x)return 0;
        if(sum + num[i]>x)
        {
            sum = num[i];
            d++;
        }
        else sum+=num[i];
    }
    return d<=m;
}
int main()
{
   cin>>n>>m;
   long long l=0,r=0;
   for(int i=1;i<=n;i++)
   {
    cin>>num[i];
     //最小的答案是单个元素的最大值
    l = max(l,(long long)num[i]);
     //最大的答案是所有元素的和
    r += num[i];
   }
//二分查找答案
   while(l<r)
   {
        long long mid = l+(r-l)/2;
        if(check(mid))r=mid;
        else l = mid+1;
   }
   cout<<l;
    return 0;
}
```
